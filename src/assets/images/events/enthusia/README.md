## Image Guidelines:
- Use high-quality images (minimum 800x600 pixels)
- Ensure images are relevant to the event type
- Keep file sizes optimized for web (under 500KB each)
- Use JPG format for photographs, PNG for graphics with transparency
- Consider the color scheme (dark theme with gold accents)

## Alternative Image Sources:
If you don't have custom images, you can use:
- Unsplash.com (free high-quality images)
- Pexels.com (free stock photos)
- Pixabay.com (free images and vectors)
